## Smart User Dashboard

A mini project built using HTML, CSS, JavaScript, Fetch API, and Node.js to demonstrate core and advanced JavaScript concepts.

## Features

User registration with validation (RegEx)
Cookies to store username
Live clock using Date & setInterval
Task manager using arrays & objects
Fetch API to load users
Search/filter functionality
Dynamic DOM manipulation

## Tech Stack
HTML, CSS
JavaScript (ES6)
Fetch API
Node.js (Express)


## Concepts Covered
Variables, Operators, Loops
Objects, Arrays, Strings
Date, Math, Boolean
RegEx & Validation
DOM & Browser
Cookies & Timing
AJAX (Fetch API)
Node.js
