
# PHP Unit Testing

## Topics Covered
- Introduction to Unit Testing
- What is Unit Testing
- PHPUnit
- PHPUnit Configuration
- Hands-on Labs / Exercises

## What I Learned
- How to test small units of code like methods and functions
- How to write PHPUnit test cases
- How to use assertions such as assertEquals, assertTrue, and assertFalse
- How to configure PHPUnit using phpunit.xml
- How to use setUp() for reusable object creation
- How to test exceptions
- How to run tests from terminal

## Files
- `src/` contains PHP classes
- `tests/` contains PHPUnit test files
- `phpunit.xml` contains PHPUnit configuration
- `screenshots/` contains proof of test execution

## Exercises Included
1. Calculator testing
2. String helper testing
3. User validator testing
