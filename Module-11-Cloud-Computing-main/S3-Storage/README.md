
AWS S3 Storage

What is S3?

Amazon S3 (Simple Storage Service) is a cloud storage service that allows users to store and retrieve data from anywhere over the internet.

Steps Performed

1. Logged into AWS Console
2. Opened S3 service
3. Created a new bucket
4. Uploaded files
5. Managed permissions for access

## Output

* Successfully created an S3 bucket
* Uploaded and stored files in the cloud
* Verified file availability
