AWS Lambda (Serverless Computing)

Objective

To understand serverless computing by creating and executing a Lambda function.

What is Lambda?

AWS Lambda is a serverless computing service that allows running code without managing servers.

Steps Performed

1. Opened AWS Lambda
2. Created a new function
3. Wrote Python code
4. Deployed and tested the function
