# SQL Practice - Student Management System

## Overview

This repository contains SQL practice files created while learning basic to intermediate SQL concepts using a Student Management System example. It includes schema creation, data insertion, and practice queries for joins, filtering, sorting, and aggregation.

## Topics Covered

* Create database and tables
* Insert, update, and delete commands
* Filtering with WHERE, LIKE, and BETWEEN
* Joins: INNER, LEFT, RIGHT, CROSS, SELF
* Sorting with ORDER BY
* Aggregation with AVG, MAX, SUM, COUNT, and GROUP BY

```bash

mysql -u root -p < queries_and_joins.sql
```
